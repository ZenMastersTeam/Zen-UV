Учебное пособие по изучению основ Zen UV.

Подготовка

Загрузка примера

  1. Убедитесь, что подключение к Интернету включено в Blender (эта опция доступна, начиная с версии Blender 4.2).

    • Нажмите на значок подключения к Интернету.
    • Разрешите доступ к Интернету.

  2. Используя examples panel выберите и загрузите пример, который называется Emergency Light Tutorial нажав кнопку Download справа от названия.

  3. Когда сцена скачается, нажмите кнопку Open, появившуюся на месте кнопки Download.

  4. Выберите объект с названием Emergency Light и перейдите в режим редактирования.

Удаление текущих UV карт и очистка свойств ребер

  1. По умолчанию Zen UV использует систему глобальной маркировки. Настроим так, чтобы Seam и Sharp маркировались одновременно.

    • Откройте вкладку Unwrap. Настройки находятся в меню на кнопке с шестерней справа от кнопки Mark by Angle.
    • Откройте меню и активируйте Mark Sharp Edges.

  2. Чтобы стереть все существующие Seams и Sharp, во вкладке Unwrap нажмите кнопку Unmark all.

  3. Проверим нет ли островов тегированных как Finished. Система Finished служит для визуального контроля за состоянием островов (окончены/не закончены) и не позволяет оператору Zen Unwrap развертывать острова если они тегированы как Finished.

    • Выделите все полигоны в модели.
    • На вкладке Unwrap откройте панель Finished и нажмите кнопку Tag Unfinished.

  4. Сотрите возможные имеющиеся UV координаты.

    • Перейдите во вкладку Advanced UV Maps.
    • Выделите все возможные UV карты.
    • Нажмите кнопку “-“.

Tip

Нет необходимости создавать новую UV Map. Zen UV работает с UV координатами, следит за тем, чтобы были имеющиеся UV Maps. Если таковых нет, то они будут созданы автоматически.

Начальная развёртка

  1. Сейчас, когда все подготовлено, мы можем начинать развертку. Этап первый – это разделение модели на UV острова. На этом этапе не имеет значения, насколько эти острова корректно развернуты. Мы только разделяем. Произведем разметку швов в зависимости от остроты рёбер. Более острые edges будут отмечены как швы и тем самым определят места где модель будет разделена.

    • Во вкладке Unwrap нажмите кнопку Mark By Angle.

  2. Почти все операции в Zen UV могут быть выполнены в 3D View, но нет более комфортного способа взаимодействовать с UV Maps чем использовать UV Editor. Для того чтобы быстро открыть UV Editor, воспользуйтесь виджетом в 3D View.

    • Нажмите кнопку расположенную слева по центру 3D View viewport с буквой “T”. В случае если UV Editor уже открыт, по нажатию на эту кнопку он закроется. Open UV Editor area

  3. Мы создали швы в зависимости от остроты рёбер (edges), но еще не разделили модель на острова. Для разделения используем оператор Zen Unwrap работающий в зависимости от того, что выделено. Так что на данном этапе убедитесь, что ничего не выделено. В режиме когда ничего не выделено Zen Unwrap просто разделяет модель на UV острова по существующим seams. Если что-то выделено, результат будет иным.

    • Убедитесь, что ничего не выделено.

    • Нажмите Zen Unwrap в панели Unwrap.

Другие режимы работы Zen Unwrap мы рассмотрим в течение этого туториала.

Включаем текстуру для проверки

  1. На этом этапе мы имеем базовое разделение модели на острова, но этого недостаточно. Нужно визуально проверить модель, создать дополнительные острова и в процессе расправить уже существующие острова, если это необходимо. На этом этапе важно улучшить восприятие атрефактов для того, чтобы выполнить работу как можно быстрее. Одним из самых действенных методов является назначение специальной текстуры, состоящей из квадратов. Если вы видите на модели искривленный квадрат, значит, в этом месте есть проблемы с разглаживанием острова. Наложим эту специальную текстуру используя Zen UV Checker.

    • На панели UV Checker нажмите кнопку Checker Texture.

  2. Цветные текстуры созданы также и для определения места положения острова на текстуре. Но для нашего случая это не суть важно. Так переключим текстуру на монохромную. Это уменьшит количество информации для восприятия, а значит уменьшит усталость от работы..

    • Откройте drop-down меню ниже кнопки Checker Texture и выберите текстуру с названием ‘zen-mono-2048x2048’ texture image

  3. Следующий шаг не важен для этой модели, но может пригодиться в будущем. Выключим отображение всех типов edges кроме seam. Эти свойства являются копией нативной системы отображения, но теперь они в близком доступе на панели UV Checker. Помним об усталости от переизбытка информации.

    • В Blender draw system отключите свойства отображения Crease, Sharp, Bevel. Оставьте включенным только отображение Seams.

  4. Текстуры для проверки обычно выглядят довольно агрессивно и мешают восприятию островов в UV Editor. Zen UV имеет возможность затемнить текущую текстуру для комфортного восприятия.

    • Используйте оператор Darken Image находящуюся в панели UV Checker основной панели аддона в UV Editor.

    Tip

    DarkenImage также можно найти в настройках оверлея в UV Editor.

  5. Для этого туториала можно просто удалить текстуру из UV Editor, поскольку в данном случае она не имеет практического назначения..

    • Найдите строчку с названием текстуры сверху рабочей зоны UV Editor и нажмите кнопку с крестиком.

Развёртка

Предварительный этап подготовки завершен. Переходим к развертке модели. Работу над чем-либо удобно разделять на части. Наша модель состоит из нескольких деталей. Так что условно разделим на этапы соответствующие каждой детали.

Развёртка внутренней цилиндрической части

  1. Начнём с цилиндрической детали.

    • В UV Editor полностью выделите цилиндрическую деталь.
    • Нажмите кнопку Isolate Island с панели Select.

    Tip

    Этот оператор работает как переключатель. Это значит что если нажать его сейчас еще раз, то все скрывшееся появится снова. Он функционирует одинаково как для 3D View, так и для UV Editor.

    Основной оператор для создания островов в аддоне Zen UV это Zen Unwrap. Это многофункциональный контекстный оператор. Он выполняет разные задачи в зависимости от следующего:

    • Какой режим выделения активен сейчас.
    • Что именно выделено на момент его запуска.
    • Какие дополнительные настройки включены.

    Zen Unwrap отслеживает три основных режима выделения.

    • Faces
    • Edges
    • Verices

    В случае если Blender находится в режиме выделения Faces, будет создан новый остров из выделенных полигонов, а ребра по краю выделенных полигонов будут маркированы как seams.

    • Выделите торец и нажмите кнопку Zen Unwrap из палитры Unwrap

    В процессе будет создано еще несколько островов. Они могут быть не корректными или не такими как в представленном видео, но сейчас это не важно.

  2. Используем режим выделения Edges для развертывания следующего острова.

    • Переключите Blender в режим выделения Edges. В этом режиме Zen Unwrap создаст seams для выделенных ребер и выполнит развертку.
    • Выделите одно ребро на длинном цилиндре и нажмите Zen Unwrap.

    Tip

    Места в которых будут созданы seams, обычно могут быть слишком заметны на текстуре и потребуют дополнительной маскировки. Чтобы не тратить это время, старайтесь создавать разрывы в мало заметных местах. Поэтому в нашем случае рекомендуется выделить какое-то из нижних ребер, поскольку оно точно будет закрыто внешней частью модели.

    Как видите Zen Unwrap точно придерживается ваших указаний и делает развертку островов несмотря на то что мы изменяем тип выделения. Для этого даже не нужно открывать его настройки. В результате мы получили корректно развернутый самый большой остров этой детали..

  3. Развернем второй торец детали.

    • Переключите режим выделения Blender в faces.
    • Выделите торец.
    • Нажмите Zen Unwrap.

  4. При использовании Mark by Angle были созданы нежелательные seams что разделило модель на большее количество островов, чем нужно. Чтобы исправить это снова воспользуемся оператором Zen Unwrap.

    • Виділіть всі полігони найбільшого циліндра та натисніть Zen Unwrap. Як результат всі виділені острови об’єдналися в один, і край виділених полігонів був маркований як seams.

  5. На этом этапе мы закончили создание островов, но многие острова остались деформированными. Можно переключиться в режим выделения edges и выполнить те же действия как для самого большого цилиндра, но острова, имеющие несколько более сложную форму, останутся недостаточно разровненными. Используем оператор Quadrify выравнивающий в ровную решетку острова созданные из квадов.

    • Переключиться в режим выделения edges.
    • Выделите один продольный edge и нажмите кнопку Quadrify в палитре Transform.

    Tip

    Подобное использование оператора Quadrify является самым быстрым, но не всегда оптимальным. Если не указать оператору, где именно нужно сделать разрыв, место разрыва острова будет создано автоматически и, возможно, не в оптимальном месте. Для полного контроля создания разрыва используйте опцию Use Selected Edges. Также вы можете создать разрыв к использованию Quadrify, используя другие методы.

  6. Выполним некоторые настройки оператора.

    • Откройте панель настроек оператора.
    • Активируйте опцию Use Selected Edges. Это укажет оператору, что нужно сделать разделение острова по выделенному ребру.

    • Выполните те же действия для всех оставшихся островов.

  7. Сейчас некоторые острова в UV Editorе могут перекрывать другие. Для того чтобы убедиться в корректном развертывании каждого острова, выполним упаковку. Это улучшит визуальное восприятие всех островов..

    • Нажмите кнопку Pack Islands с панели Pack.

  8. Вскоре мы продолжим работать над другими деталями, но сперва используем систему Finished. Это сделает возможным визуально контролировать, какие острова уже развернуты, и защитит готовые острова от повторного развертывания.

    • Перейдите на вкладку Unwrap.
    • Разверните панель, которая называется Finished.
    • Выделите все острова.
    • Нажмите кнопку Tag as finished с панели Finished

Tip

Острова тегирующиеся Finished сортируются и перемещаются правее UV Area. Это поведение по умолчанию, но это можно отключить если отключить опцию Auto Sort Islands находящуюся правее оператора Sort Islands by Tags в диалоге с кнопкой с шестерней.

Развёртка внешней части

  1. Переходим к следующей детали. Для этого необходимо показать все детали модели.

    • Нажмите кнопку Isolate Island с панели Select

    • Выделите часть модели, которая выглядит как внешняя часть.

    • Нажмите кнопку Isolate Island снова. Теперь мы видим видимую только нужную нам деталь. Остальные детали спрятаны и они не будут мешать.

  2. Эта деталь имеет четкие контуры, корректные seams, но некоторые острова можно сделать лучше. Внутренняя часть представляет собой цилиндр, который может быть представлен одним островом вместо нескольких. Создадим один остров из внутренней части.

    • Выделите всю внутреннюю часть кроме острова, который представляет собой многогранный полигон.
    • Нажмите Zen Unwrap.

  3. Созданный остров может выглядеть лучше если выровнять его в ровную решетку. Используйте оператор Quadrify. Обратите внимание. Если у вас активирована опция Use Selected Edges, то лучше перейти в режим выделения faces. Ведь в режиме выделения edges оператор Quadrify разорвет остров по выделенным edges. Также обратите внимание на то, что Zen UV все время пытается вам помочь и для того, чтобы указать оператору с каким островом вы хотите работать, не обязательно выделять весь остров. Достаточно выделить один или несколько полигонов.

    • Выделите один или несколько полигонов в острове внутренней части текущей детали и нажмите Quadrify.

  4. Следующий остров слишком велик, замкнут и вывернут. Для того чтобы избавиться от этих недостатков, разделите его на несколько островов меньшего размера.

    • Выделите боковые ребра.
    • Нажмите Zen Unwrap.

  5. Остров представляющий верхний торец разложен корректно, но можем сделать его более компактным. Также мы должны следить за тем, чтобы граничные edges любого острова были максимально выровнены по осям. Это сделает разрывы на текстуре гораздо менее заметными.

    • Выделите один или несколько faces верхнего острова.
    • Нажмите Select Island с панели Select.
    • Перейдите в вид сверху и используйте оператор Project From View, который создаст ровный остров из выбранных полигонов.

  6. Разделение на острова завершено. Выполните Pack и Tag Finished как было описано для предыдущей детали.

  7. Нажмите Isolate Island, чтобы показать все детали и подготовиться к следующему этапу туториала.

Развертка мелких частей

  1. Изолируем мелкие детали для более комфортной работы.

    • Выделите все мелкие детали.
    • Нажмите кнопку Isolate Island

  2. На всех мелких деталях оператор Mark by Ange сработал корректно, кроме цилиндрических деталей. Давайте исправим это.

    • Выделите по одному ребру на каждом из цилиндров и нажмите Zen Unwrap.

  3. Перейдите к детали скрепляющей провода. Она была разделена на несколько частей, что не является оптимальной разверткой. Достаточно отделить торцы, а остальное оставить одним островом..

    • Выделите всю деталь.
    • Снимите выделение с торцов.
    • Нажмите Zen Unwrap.

  4. Мы создали сплошной остров, но его развертка не корректная. Нужно разорвать и разровнять радиальный остров.

    • Выделите пару ребер, где нужно разделить.
    • Нажмите Zen Unwrap.

  5. На этом этапе мы получили корректное деление на острова, но некоторые острова имеют растяжение. Некоторые другие острова имеют не оптимальную форму. Нужно выделить и выровнять все состоящие из квадов острова.

    • Используйте оператор Select Quaded Islands с панели Select.

    • Используйте оператор Quadrify чтобы выровнять все выделенные острова в ровную решетку.

  6. Выполните Pack и Tag Finished для завершения развертки мелких деталей.

  7. Нажмите Isolate Island, чтобы показать все детали и подготовиться к следующему этапу туториала.

Развёртка проводов

  1. Изолируем провода для более комфортной работы.

    • Выделите все провода.
    • Нажмите Isolate Island, чтобы скрыть все лишние детали.

  2. Видно что сейчас каждый провод разделен на несколько островов. Сделаем так, чтобы каждый провод был представлен ​​одним островом. Выделите все полигоны и нажмите Zen Unwrap.

  3. Мы получили разделение где каждый провод один остров, но развертывание не корректное. Давайте развернем провода самым быстрым способом.

    • Убедитесь, что выделены все острова.
    • Используйте оператор Quadrify чтобы выровнять каждый остров в ровную решетку.

    Tip

    В случае развертки проводов или других подобных моделей очень удобно пользоваться опцией Use Selected Edges оператора Quadrify, которую мы рассматривали ранее. Таким образом мы получим разрывы именно там, где нужно с первой попытки.

  4. В завершение упаковаем для лучшего визуального контроля и маркируем острова как завершенные (Finished).

    • Выполните Pack.
    • Выполните Tag Finished для завершения развертывания проводов.

  5. Нажмите Isolate Island, чтобы показать все детали и подготовиться к следующему этапу туториала.

Развёртка резинового крепления

  1. Изолируем крепления для более удобной работы. Давайте используем кое-что из Finished системы.

    • Нажмите Isolate Island для отображения всех частей.
    • Нажмите Select в панели Finished.
    • Нажмите кнопку Hide в панели Finished.

  2. Эта деталь состоит из нескольких островов, что не является оптимальным решением. Давайте создадим один остров вместо нескольких.

    • Выделите все принадлежащие креплению острова.
    • Снимите выделения с торцов, которые выглядят как шестиугольники.
    • Нажмите Zen Unwrap.

  3. Остров создан в результате последних действий, имеет не корректную развертку. Давайте исправим это.

    • Выделите edges принадлежащих отверстию и edges принадлежащих внутреннему краю крепления.
    • Нажмите Zen Unwrap

  4. Теперь у нас есть развернутый остров. Присутствие в нем треугольных faces не позволяет использовать оператор Quadrify для максимальной разглаживания. Рассмотрим на примере этого острова работу оператора Reshape Island. Суть работы Reshape Island заключается в том, что он разравнивает выделенные лупы ребер в ровную линию, а остальные вершины расслабляет.

    • Выделите полностью ребра проходящие продольно через центр острова.
    • Нажмите Reshape Island для выравнивания острова.

  5. На этом развертку острова можно считать завершенную. Давайте маркируем его как завершенный.

    • Нажмите Pack.
    • Выделите все острова.
    • Маркируйте с помощью Tag Finished для завершения развертки крепления.

  6. Нажмите Isolate Island, чтобы показать все детали и подготовиться к следующему этапу туториала.

Паковка

Предварительный этап создания островов и развертки завершен. Пришло время разместить острова на текстуре таким образом, чтобы осталось как можно меньше свободного места. Все свободное место на текстуре – это нерациональное использование ресурсов.

Финальная паковка

  1. Убедитесь, что не осталось скрытых деталей. И выполните паковку.

    • В 3D View в главном меню нажмите Mesh - Show/Hide - Reveal Hidden.
    • Выполните упаковку используя оператор Pack.

Уменьшение свободного места с помощью стаков (оверлапов)

  1. На этом этапе есть плотно упакованные детали и работу можно считать завершенной. Но если проанализировать конструкцию модели, можно заметить, что некоторые острова выглядят похоже и даже являются копией друг друга. Например, вырезы под болты, сами болты, некоторые мелкие детали и т.д. Для экономии места на текстуре одинаковые острова можно сложить один на другой. С точки зрения создания UV координат этот процесс называется Stacking.

    • Перейдите на вкладку Stack.
    • Нажмите кнопку Replicas. Теперь есть выделенные острова, которые имеют копии. Эти острова называются Replicas. Причем, их прототипы остались не выделенными.

    • Переместите выделенные острова в сторону от UV Area. Теперь у UV Area остались только уникальные острова не имеющие копий (Singles), и острова, которые являются прототипами (Primaries).

  2. В этом туториале мы используем Blender Pack Engine. Следующие шаги для стаковки будут описаны именно для этого паковщика.

    • Спрячьте острова, которые были перемещены в сторону.

    • Установите корректный (желательный) Texture Size в палитре Pack. Для этого туториала размер текстуры не имеет значения, но возможно вы планируете создать текстуру по другому туториалу.

    • Установите необходимую величину Margin что обеспечит корректный вид текстуры при отдалении камеры.

    • Выполните повторную упаковку используя кнопку Pack.

    • В панели Stack переключите систему в режим Selected.
    • Для того чтобы показать скрытые Replicas в 3D View в главном меню нажмите Mesh - Show/Hide - Reveal Hidden.
    • Выделите всё кроме Replicas.
    • Нажмите Stack для того, чтобы стаковать не выделенные Replicas на места выделенных Primaries.

Паковка закончена. Осталось несколько финальных шагов.

Установка острых рёбер (Sharp Edges)

  1. Корректно установленные sharp edges это не только залог хорошего внешнего вида модели, но также залог запекания normal map без ошибок.

    • Используйте оператор Sharp by UV Borders. Этот оператор маркирует ребра, которые являются граничными ребрами островов как sharp.

    • В настройках оператора включите Unmark sharp. Это сотрет все предварительно установленные sharp edges.

    • Нажмите кнопку Smooth by Sharp. Это установит сглаживания полигонов в соответствии с sharp edges.

Теперь модель Emergency light имеет корректные и оптимизированные UV координаты и полностью готова для создания текстуры и запекания normal map.
