# Version 5.2.0

## **General**

- Added support for [**Blender v5.0**](https://developer.blender.org/docs/release_notes/5.0/)
- New operators  
- Fixes  

## **Pack**

- Optimized the **Pack** operator.

## **Unwrap**

- Added the ability to create default settings in the **Auto Unwrap** operator.  
- Disabled **Auto Unwrap** functionality when UV Sync Selection is Off.  
- Added the ability to **Copy/Paste UV Maps** in Object Mode.

## **Select**

- Added the **Select Cylinder Cap** operator for selecting caps of cylindrical mesh parts.  
- Added the **Select Cylinder Faces** operator for selecting sides of cylindrical mesh parts.  
- Added an operator for selecting edges between different materials.  
- In the **Zen Sync** operator for [**Blender v5.0**](https://developer.blender.org/docs/release_notes/5.0/), added the ability to synchronize in any selection mode (Vert, Edge, Face).

## **UV Checker**

- Optimized the filter when selecting a texture for orientation checking.

## **Trimsheet**

- Disabled the ability to delete **Trims** in **Trim Transform** mode.
